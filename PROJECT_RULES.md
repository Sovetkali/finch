# Project Rules

## Language

- Always respond in Russian.
- Write code and code comments in English.

## Skills

- When creating or updating a Codex skill for this project, add the usage command to `CHEATSHEET.md`.
- Keep the cheat sheet up to date with new local-development and release workflows.

## Working style

- Prefer narrow, concrete changes.
- Keep production release changes separate from local-only helper files.
